# Model and Cost Function

## 符号定义
对于回归问题, 假设有一堆训练集, 记: <br/>
$m$ - 训练集的样本数量 <br/>
xs - 输入变量/特征 <br/>
ys - 输出变量/目标变量 <br/>
(x, y) - 一个样本对 <br/>
(x(i), y(i)) - 第i个样本对 <br/>

$`\sqrt{2}`$
